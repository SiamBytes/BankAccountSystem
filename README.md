# Bank Account System (Java)

A console-based Java program for simple banking operations.

## Features
- Create bank accounts
- Deposit money
- Withdraw money
- Check balance
- Menu-driven console interface

## How to Run
1. Compile:
   ```bash
   javac src/BankAccountSystem.java
   ```
2. Run:
   ```bash
   java -cp src BankAccountSystem
   ```
